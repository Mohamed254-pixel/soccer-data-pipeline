# Soccer Data Pipeline

An end-to-end data engineering project that collects, transforms, and analyzes soccer data using Python and SQL.

## Tech Stack
- Python
- SQL
- Git
- GitHub

## Project Goals
- Build an automated data pipeline
- Store soccer data in a database
- Perform analytics using SQL
- Practice data engineering concepts
