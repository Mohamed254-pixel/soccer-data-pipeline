# Soccer Data Pipeline

## Overview
Built a soccer data pipeline using Python, Pandas, MySQL, and SQL to process and analyze international football match data.

## Dataset
- 43,281 international football matches
- Historical data from 1872 to present
- 9 columns including teams, scores, tournament, city, and country

## Technologies
- Python
- Pandas
- MySQL
- SQL
- Jupyter Notebook
- Git
- GitHub
